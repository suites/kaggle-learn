# kaggle-learn

Summary and notebooks of my Kaggle Learn courses, including hands-on exercises and key insights.

## Installation

```bash
# Install uv
brew install uv

# Install dependencies
uv sync

# Set up Jupyter kernel
uv run python -m ipykernel install --user --name=kaggle-learn
```

## Getting Started

```bash
# Run Jupyter notebook
uv run jupyter notebook
```
